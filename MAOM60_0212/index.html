Fontosabb Git parancsok

  • git init: Inicializálja a lokális Git repository-t egy új projekt számára
  • git clone: Letölti egy meglévő távoli repository tartalmát a lokális gépre
  • git add: Hozzáadja a fájlt a “staged” állapothoz, hogy később commitolni lehessen
  • git commit -m: : Commitolja a változtatásokat a lokális repository-ban
  • git remote: A Git eltárolhat bizonyos információkat
  • git push: Feltölti a lokális változtatásokat a távoli repository-ba
